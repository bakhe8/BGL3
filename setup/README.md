# ⚠️ ملف README مهم

## حول مجلد Setup

هذا المجلد **منفصل تماماً** عن البرنامج الأساسي ويُستخدم **لمرة واحدة فقط** لاستيراد البيانات الأولية.

## الاستخدام

1. افتح: `http://localhost:8000/setup/views/import.php`
2. ارفع ملف CSV يحتوي على:
   - `supplier_name` (أسماء الموردين)
   - `bank_name` (أسماء البنوك)
3. راجع البيانات في صفحة المراجعة
4. أكّد واحفظ
5. **احذف هذا المجلد بالكامل** بعد الانتهاء

## البنية

```
setup/
├── database/
│   ├── import.sqlite     ← قاعدة بيانات مؤقتة (تُحذف بعد الانتهاء)
│   └── schema.sql
├── views/
│   ├── import.php        ← صفحة رفع CSV
│   ├── review.php        ← صفحة المراجعة
│   └── complete.php      ← صفحة النجاح
├── api/
│   ├── upload.php        ← رفع ومعالجة CSV
│   ├── get-suppliers.php
│   ├── get-banks.php
│   ├── update-supplier.php
│   ├── update-bank.php
│   ├── migrate.php       ← نقل البيانات للقاعدة الأساسية
│   └── cleanup.php       ← حذف الملفات المؤقتة
├── templates/
│   └── sample.csv        ← ملف نموذجي
└── assets/
    └── setup.css

```

## ملاحظات مهمة

- ✅ **معزول تماماً** - لا يؤثر على البرنامج الأساسي
- ✅ **آمن** - يمكن حذفه في أي وقت
- ✅ **مؤقت** - يُستخدم مرة واحدة فقط
- ⚠️ **احذفه بعد الانتهاء** - لا حاجة له بعد نقل البيانات

## الحذف

بعد نقل البيانات:
```bash
# يمكنك حذف المجلد بالكامل
rm -rf setup/
```

أو استخدم زر "حذف ملفات الإعداد" في صفحة الإكمال.

# Input Folders

ضع ملفاتك هنا للمعالجة الدفعية:

## Excel Files
ضع ملفات Excel (.xlsx, .xls, .csv) في: `excel/`

مثال:
```
excel/
  ├── suppliers.xlsx
  ├── banks.xlsx
  └── combined.csv
```

## Word Files
ضع ملفات Word (.docx) في: `word/`

مثال:
```
word/
  ├── contract_001.docx
  ├── contract_002.docx
  └── guarantees.docx
```

## ملاحظة
سيتم معالجة جميع الملفات في المجلدات تلقائياً عند الضغط على "معالجة الكل".
